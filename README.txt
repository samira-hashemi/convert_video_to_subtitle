
موضوع پروژه :تولید زیرنویس برای ویدیو
برای اجرا پروژه بعد از دانلود پوشه از github 
 در پوشه یک محیط مجازی ایجاد کنید 
و بعد با دستور pip install -r requirements.txt پکیج های داخل این فایل را نصب کنید 
در مرحله بعد در خط فرمان vscode در مسیر پوشه قرار بگیرید
 و با دستور flask run   یا python app.py پروژه را اجرا کنید
مراحل انجام کار :ابتدا در صفحه وب روی کلید  choose file کلیک میکنید
 و و فایل ویدیو مورد نظر را انتخاب میکنید وبعد روی کلید convert کلیلک میکنید
 منتظر میمانید در همان صفحه ویدیو همراه با زیرنویس نمایش داده شود
